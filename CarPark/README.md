Чем field отличается от property? Они оба относятся к инкапсулированию, но property может возвращать и менять данные.

Зачем нужны методы? Для того чтоб управлять свойствами класса, для создания функций классу.

Почему List<Car>? List нужен для хранения объектов. В C# нельзя хранить объекты внутри Array как в js.

Тут вполне можно без валидации.
